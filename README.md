# ecommerce-swing
Este projeto foi desenvolvido como trabalho de conclusão de curso (TCC) com o objetivo de simular um sistema de e-commerce completo, utilizando Java com interface gráfica baseada em Swing.
A proposta foi criar uma aplicação que representasse, de forma prática, o funcionamento de um comércio digital, indo além da parte visual e explorando também a lógica de gerenciamento por trás de uma loja.
